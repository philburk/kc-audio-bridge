# KC Audio Bridge

Cross-platform audio output and input for Kotlin/Compose apps on Android, Web, and Desktop.

This is a Kotlin/Compose Multiplatform library that can output and input audio on
almost every device.
It provides a simple and portable API that can be used to play or record stereo floating point buffers.

The API is defined in [AudioBridge.kt](https://github.com/philburk/kc-audio-bridge/blob/main/composeApp/src/commonMain/kotlin/com/mobileer/audiobridge/AudioBridge.kt).

To try a simple demo on the web, visit: https://transjam.com/kc-audio-bridge/
